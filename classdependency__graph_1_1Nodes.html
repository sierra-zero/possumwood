Detailed Description

Data structure holding node instances. Iterators are not guaranteed to remain valid after operations, but the node instances are stored in a vector container (their positions in the array will not change after add, and erase shifts subsequent indices), and each node instance's memory address is guaranteed not to change during its lifetime (Nodes are stored as pointers).
